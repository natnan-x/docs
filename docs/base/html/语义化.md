# HTML 语义化标签

**定义：**
HTML 语义化标签是指能表达内容含义的标签，让浏览器、开发者和搜索引擎都能清楚地理解页面结构和内容类型。

## 一、语义化的定义

语义化（Semantic）就是用恰当的标签表达内容的意义，而不仅仅是为了视觉样式。

- **非语义化标签：** `<div>`、`<span>`
- **语义化标签：** `<header>`、`<nav>`、`<article>`、`<footer>` 等。

## 二、常见语义化标签

| 标签                        | 含义                          | 示例场景               |
| --------------------------- | ----------------------------- | ---------------------- |
| `<header>`                  | 页头，通常包含 logo、导航栏等 | 页面顶部导航区         |
| `<nav>`                     | 导航链接区域                  | 主导航、侧边导航       |
| `<main>`                    | 页面主体内容                  | 页面主要内容部分       |
| `<section>`                 | 页面中某个主题区块            | 新闻模块、产品模块     |
| `<article>`                 | 独立的可复用内容              | 文章、博客、帖子       |
| `<aside>`                   | 侧边栏内容                    | 推荐内容、广告栏       |
| `<footer>`                  | 页脚内容                      | 版权信息、联系信息     |
| `<figure>` / `<figcaption>` | 插图及说明文字                | 图片及说明文字组合     |
| `<time>`                    | 表示时间或日期                | 发布时间、更新时间     |
| `<mark>`                    | 高亮显示文本                  | 搜索结果关键字标记     |
| `<address>`                 | 表示联系信息                  | 公司地址、作者联系方式 |
| `<details>` / `<summary>`   | 可展开或折叠的内容            | 折叠式 FAQ、说明文字   |
| `<progress>`                | 进度条                        | 文件上传、任务进度     |
| `<meter>`                   | 度量值                        | 电量、分数、比例       |
| `<blockquote>`              | 块级引用内容                  | 引用文章段落           |
| `<cite>`                    | 引用作品标题                  | 书籍、文章、电影标题   |
| `<abbr>`                    | 缩写词定义                    | HTML、CSS 等缩写说明   |
| `<code>` / `<pre>`          | 代码片段与格式化代码块        | 展示示例代码           |
| `<em>` / `<strong>`         | 强调与重点文字                | 内容强调、重点突出     |

## 三、语义化的好处

1. **利于 SEO：** 搜索引擎更容易理解页面结构，提高排名。
2. **可访问性：** 辅助设备（如屏幕阅读器）可更准确地读取内容。
3. **易维护：** 结构清晰，开发者阅读代码更直观。
4. **样式分离：** 结构与表现分离，配合 CSS 更灵活。

## 四、常见错误示例

```html
<!-- ❌ 非语义化 -->
<div id="header">
  <div class="nav">首页 | 产品 | 联系</div>
</div>

<!-- ✅ 语义化 -->
<header>
  <nav>
    <a href="/">首页</a>
    <a href="/products">产品</a>
    <a href="/contact">联系</a>
  </nav>
</header>
```

## 五、总结（面试高频一句话）

> HTML 语义化是指使用具有含义的标签来描述页面结构，使代码更易读、利于 SEO 与可访问性。
